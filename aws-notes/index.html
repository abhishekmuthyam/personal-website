Backend Developer Map

These notes focus on AWS services that commonly appear in backend projects, interviews, and deployment workflows.

Compute

  • EC2 for virtual servers
  • Lambda for serverless functions
  • Elastic Beanstalk for simple app deployment

Storage and Database

  • S3 for object storage
  • RDS for relational databases
  • DynamoDB for key-value and document data

Operations

  • IAM for users, roles, and permissions
  • CloudWatch for logs and metrics
  • API Gateway for managed API entry points
